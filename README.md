# Cognizant Digital Nurture 5.0 – Deep Skilling (Java FSE)

## Overview

This repository contains my solutions, hands-on exercises, and projects completed as part of the **Cognizant Digital Nurture 5.0 – Deep Skilling Program** for the **Java Full Stack Engineer (FSE)** learning track.

The program is designed to strengthen software engineering fundamentals, backend development, frontend development, cloud concepts, DevOps practices, and modern application development using Java and React.

---

## Learning Objectives

- Understand Software Engineering Principles and Design Patterns.
- Strengthen problem-solving skills using Data Structures and Algorithms.
- Develop enterprise applications using Spring Framework and Spring Boot.
- Build RESTful APIs and Microservices.
- Learn database programming using SQL and PL/SQL.
- Apply Test-Driven Development (TDD) using JUnit and Mockito.
- Develop responsive user interfaces using React.
- Understand Version Control, CI/CD, Docker, and Cloud Fundamentals.
- Explore Generative AI fundamentals and modern software development practices.
